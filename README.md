# Proyecto de Consola: Fundamentos de JavaScript

Bienvenida: Este proyecto consiste en una aplicación interactiva de maqueta básica que se ejecuta directamente desde la consola del navegador. El objetivo principal es demostrar el uso correcto de la sintaxis básica, estructuras de control, modularización con funciones y la manipulación avanzada de arreglos y objetos para una consola básica y fácil de interpretar para cualquier usuario.

---

# Características del Proyecto

La aplicación simula un sistema dinámico de cotización de servicios de diseño y desarrollo web que cumple con los siguientes requerimientos:
* **Interactividad directa:** Captura datos en tiempo real mediante el uso de `prompt()` y validación inmediata de entradas.
* **Modularización:** Separación de la lógica matemática en funciones independientes y reutilizables (`sumar`, `restar`, `multiplicar`, `dividir`).
* **Estructuras de Control:** Implementación de flujos condicionales (`if`, `else`, `switch`) y ciclos de repetición (`for`, `while`) para procesar presupuestos y flujos lógicos de compra.
* **Estructuras de Datos:** Almacenamiento y gestión de datos complejos combinando arreglos dinámicos y colecciones de objetos con métodos integrados.

---

# Estructura del Código por Lecciones

El desarrollo fue realizado de forma progresiva siguiendo la guía estructurada del módulo:

1. **Lección 1 (Introducción):** Configuración inicial del entorno, uso de `console.log()` y capturas personalizadas con `prompt()` y `alert()`.
2. **Lección 2 (Variables y Condicionales):** Declaración de variables con `let` y `const`, operaciones aritméticas y lógica condicional con `if/else` y `switch`.
3. **Lección 3 (Arreglos y Ciclos):** Control de inventario/precios usando colecciones indexadas, bucles indexados y condicionales (`for`/`while`), además de filtros dinámicos con `.filter()`.
4. **Lección 4 (Funciones):** Abstracción y encapsulado de operaciones lógicas. Creación de una función optimizada que coordina subtotales e impuestos llamando a funciones secundarias.
5. **Lección 5 (Objetos):** Creación de un catálogo estructurado mediante un arreglo de objetos, usando la palabra clave `this` en métodos internos y recorridos óptimos con `.forEach()`.

---

# Cómo Ejecutar la Aplicación

1. Clona este repositorio o descarga los archivos en tu equipo.
2. Abre el archivo `index.html` en cualquier navegador web.
3. Activa las Herramientas de Desarrollador presionando la tecla **F12** (o haz clic derecho e ingresa a *Inspeccionar*).
4. Dirígete a la pestaña **Consola (Console)** para interactuar con los prompts y visualizar el flujo de datos en tiempo real.
